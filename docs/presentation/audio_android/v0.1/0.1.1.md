## [v0.1.1] - 2026-03-08

### 技术改进/重构 (Changed/Refactor)
* 重构 `audio_android` Native 层构建方式，改为通过 `add_subdirectory()` 接入 `libs/audio_core` 与 `libs/audio_api`。
* 统一 Android Native 编码/解码链路只依赖 `bag_api.h`，不再直接包含 `bag/fsk/fsk_codec.h`。
* 统一表现层版本为 `0.1.1`，About 页面继续通过 JNI 读取 `bag_core_version()` 展示内核版本。

### 修复 (Fixed)
* 修复 Android Native 层直接依赖 core 内部头文件带来的边界耦合问题。
