## [v0.3.4] - 2026-04-24

### 新增功能 (Added)
* 新增 `Library` 分类整理 v1，为已保存音频加入应用内文件夹元数据层，支持创建、重命名、删除文件夹，并支持按 `All`、`Uncategorized` 与自定义文件夹切换列表视图，同时提供单条与批量移动音频到文件夹的入口。
* 新增面向协议解释的播放器可视化：为 `pro` 模式加入独立的解释型 visual，围绕频点命中、`symbol/nibble` 节拍与 `ASCII byte` 关系展示当前编码过程；为 `ultra` 模式加入紧凑的 `symbol` 阶梯图与 `Now/Next` 频点提示，替代原先可读性较弱的单轨包络图。
* 新增播放器底部音频信息弹窗，将编码类型、播放时长、`sample rate`、`frame size` 与 `flash voicing style` 收口到统一入口，并按“用户信息 / 技术信息”分组展示。

### 技术改进/重构 (Changed/Refactor)
* 重构播放器 visual 路由，让 mini player 与详情页统一按 `transport mode` 选择 `flash`、`pro`、`ultra` 对应的可视化实现，不再把不同协议模式混在同一条波形分支里。
* 重构 `AudioPlaybackTransportControls` 与 `ProEncodingExplanationVisualizer` 的 Kotlin 结构边界，将主控制区、信息弹窗与 `pro` 解释层级拆成独立模块，收口状态拥有者与显示协调逻辑，降低共享播放器骨架与教学型 visual 的职责混杂。
* 调整 Android 多语言口径，将语言列表中的 `English`、`简体中文`、`繁體中文`、`日本語`、`Deutsch`、`Русский`、`Español`、`Português` 固定为自称显示，并把新增的 `pro` visual、播放器信息弹窗与 `Library` 文件夹文案同步到当前维护的 8 套语言资源。

### 修复 (Fixed)
* 修复 `pro/ultra` 播放时 visual 过于抖动、密集竖线与信息遮挡的问题，将显示更新对齐到 `symbol` 节拍，并修正 `pro` 频点命中图中底部频率被压住、分组不清与单位缺失的问题。
* 修复播放器详情页顶部重复显示 `Now playing / generated audio / 时长 / 下载` 的信息冗余，改为由底部操作区统一承载下载与信息查看入口。
* 修复 Android 文案回归防线缺失的问题，新增字符串资源一致性校验与相关单测，避免新增 UI 文案时只更新英文而漏掉其他语言资源。
