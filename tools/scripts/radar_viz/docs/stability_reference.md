# 维度参照标准：链路稳定 (Stability)

在 WaveBits 音频可视化体系中，"链路稳定"等同于**物理层鲁棒性 (Robustness)**。它描述了信号在通过恶劣传输环境（如充满电磁干扰的工厂或强背景噪声）时，依然能被接收端正确还原的能力。

## 核心衡量指标：物理层鲁棒性 (Robustness)

该维度的数值由以下技术因素共同决定：

### 1. 符号时长 (Bit/Symbol Duration)
*   **高稳定 (8-10)**：拥有极长的采样窗口（如 `Litany` 的 6x 采样）。这意味着即便环境中有强烈的瞬时脉冲干扰，解码器依然可以通过大量的样本均值剔除噪声。
*   **低稳定 (1-3)**：符号极其短促。哪怕丢失几毫秒的数据，也会导致整个符号解调失败。

### 2. 同步策略 (Synchronization Strategy)
*   **强制重置 (Reset/Gap)**：如 `Litany` 在 bit 之间插入的静默槽。这种设计消除了符号间干扰（ISI），确保每一个 bit 都是从零状态开始判定，容错度极高。
*   **流式变时 (`Zeal`)**：依赖极其精准的时钟同步。一旦出现多径反射导致的时间偏移，整个数据流都会陷入相位错位。

### 3. 抗抖动能力 (Jitter Resistance)
*   **平滑切换**：频率切换平滑，不易产生频谱泄漏。
*   **人为抖动 (`Collapse`)**：为了表达"崩坏"情绪，故意在信号中引入不稳定的震颤（Tremor）。这种设定会直接破坏解码器的 PLL（锁相环）稳定性，使数值跌至冰点。

### 4. 载波选择 (Carrier Frequency)
*   **低频优势**：较低的载波频率（如 `Litany` / `Void` 的 220/440Hz）在穿透障碍物和抗衰减方面通常优于高频信号。

---

## 典型参照案例：
*   **Litany (10)**：长采样 + 间隔重置，被誉为"即使在强电磁干扰中也能听清的编码"。
*   **Pro (20/OVR)**：高阶协议的绝对稳定，使用 DTMF 双音多频物理层，抗噪性远超常规 BFSK。
*   **Collapse (1)**：设计目标即为"不可靠传输"，链路随时可能中断。
